# Refund Policy v1

Clauses:

1. **Window**: orders are refundable within 30 days of `delivered_at`.
2. **Final sale**: items in category `final-sale` are never refundable.
3. **Damaged**: items marked `damaged_on_arrival=true` are refundable
   regardless of window or category.
4. **Default**: otherwise eligible if Window is satisfied and category is
   not `final-sale`.
